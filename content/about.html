About

The Chargeverse is an expansive universe based on the Super Mario series that I've been writing since September 2024. Since then, I've written almost 70,000 words of original stories, which is about 200 pages! I have a feeling Google Docs isn't liking that so much. Anyway, I've been wanting to share my stories, so I thought I should share them here. I've also provided some extra lore documents. These aren't required to read, but for if you want some extra information on my universe. I'm also working on a ROM hack for Super Mario World, which is an event that takes place in-universe. Thanks for checking out the site!
